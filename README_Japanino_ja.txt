Funnel Server for Japaninoについて

Funnel Server for Japaninoを使う場合には、同梱されている以下の
スケッチを利用してください。

・JapaninoPOVFirmata
・StandardFirmataForJapanino

前者は、大人の科学マガジンVol.27ふろくの光残像キットを扱うための
スケッチです。

後者は、［スケッチ例］→［Firmata］→［StandardFirmata］をJapanino
用に変更したものです。

※Japaninoでは、ボードによりオリジナルのStandardFirmataの通信速度
である57600bpsでは安定しないことがあるため、38400bpsに変更して
います。
