ALLConstant
public static const ALL:uint = 0xFFFF

Represents all of the connected Fio modules. Example, to set the value of pin 10 for all connected Fio modules: fio.module(All).pin(10).value = x.

全てのモジュールを表します。fio.module(ALL).pin(10).value = xのようにすることで、全モジュールの10番目のピンの値をxに設定します。
