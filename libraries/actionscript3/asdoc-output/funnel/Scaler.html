Classpublic class Scaler
InheritanceScaler Object
Implements IFilter

Scales up an input value from its min and max range to a specified minimum to maximum range. A number of scaling functions are provided.

ある範囲の入力をある範囲にスケーリングするためのクラスです。直線でのスケーリング以外に、よく使われるカーブも用意されています。
