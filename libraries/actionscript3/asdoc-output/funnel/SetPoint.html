Classpublic class SetPoint
InheritanceSetPoint Object
Implements IFilter

Divides an input to 0 or 1 based on the threshold and hysteresis. You can also use multiple points by providing a nested array such as [[0.4, 0.1], [0.7, 0.05]].

アナログの値に対して閾値とヒステリシスを持つポイントをセットし、現在の状態を段階化して返します。ポイントが1つの場合の出力は0または1の2種類、ポイントが2つの場合は0または1または2の3種類、ポイントがn個の場合は0からnまでのn種類になります。
