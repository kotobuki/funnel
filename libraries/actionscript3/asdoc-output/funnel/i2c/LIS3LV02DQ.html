Classpublic class LIS3LV02DQ
InheritanceLIS3LV02DQ I2CDevice Object
Implements flash.events.IEventDispatcher

This is the class to express a LIS3LV02DQ 3-axis accelerometer Jeff Hoefs 4/9/09 revised 3/5/2011 to expose acceleration values in Gs and allow user to choose from +-2G (default) or +-6G dynamic ranges
